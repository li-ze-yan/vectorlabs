<h2>🛠️ Installation Steps:</h2>

<p>1. 1. Clone the repository</p>

```
git clone repo_url
```

<p>2. 2. Install npm modules</p>

```
pnpm install
```

<p>3. 3. Start development server using below command</p>

```
pnpm run dev
```
